2. What is SQL? MySQL? What are relational databases? 
Structured query language, is a language used to work and change databases for example inserting and updating data. MySQL is like a version of SQL its a relational database system. Relational databases are databases that store data into tables and use certain keys to link tables together
3. What is normalization in SQL? 
Normalization makes the data stored more efficient by not having the need to copy the same code and having relationships with different tables.
4. What are primary and foreign keys? 
Primary keys are need for each table that is created as they are giving the table its own id. A foreign key is where a table is making use of another tables primary key in order to link them together and share data.
